Vercel serverless filesystem is read-only. Tariff updates must be applied locally then deployed. 

# 1. Dry-run to preview diff
npm run data:upload-xlsx -- --file=data/new-tariff.xlsx --dry-run

# 2. Apply + create version entry
npm run data:upload-xlsx -- --file=data/new-tariff.xlsx --label=v2026-07 --replace

# 3. Commit + push → Vercel auto-deploy
git add data/tax.json data/versions/
git commit -m "deploy(data): update tariff v2026-07"
git push origin main
Tariff versions stored in data/versions/
